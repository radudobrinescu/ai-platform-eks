1. EBS Data Volume Snapshot

Pre-pulled 13 GiB Ray LLM image baked into EBS snapshot. Karpenter boots GPU nodes with image already on /dev/xvdb.

Automated by Terraform

2. SOCI Lazy-Loading

Bottlerocket containerd streams image layers on demand. Graceful fallback when snapshot is stale or missing.

Automated by Terraform

3. S3 Model Weight Cache

initContainer syncs HF weights from S3 (~15s) instead of downloading from HuggingFace (~60s). Auto-populated by sidecar after first load.

Auto-warms after first deploy

4. GPU Warm Pool

One g6.xlarge node kept always-on with Ray LLM image cached. Low-priority placeholder evicted instantly by real workloads.

Always active (ArgoCD)
